# 오늘장(ONULJANG) — 도매시장 실시간 경매로 잇는 산지·소비지 가격 코파일럿

**제11회 「농업·농촌 공공데이터+AI」 활용 창업경진대회 / 제품·서비스 개발 트랙 제출 기획서**

> 한 줄 요약 — **하나의 도매시장 실시간 경매 데이터를 양방향으로 번역한다.** 농가에게는 "오늘 어느 시장에 보내야 운송비 빼고 가장 많이 남는가(실수령 라우팅)", 소비자에게는 "오늘 무엇이 평년보다 싸고 비싼가(장보기 신호등)". 산지와 소비지를 한 화면에서 잇는 유일한 서비스.

---

## 1. 문제 정의 (Why now)

대한민국 농산물 가격은 **같은 품목·같은 등급이라도 전국 32개 공영도매시장마다 따로 형성**된다. 그런데:

- **농가**는 늘 가던 한두 시장(주로 가락)에 관성적으로 출하한다. 같은 날 다른 시장이 더 비싸도, 운송비를 빼면 어디가 유리한지 계산이 어려워 **며칠·수십만 원 단위로 손해**를 본다. 농가 인구의 **55.8%가 65세 이상**(2024 통계청)이라 복잡한 가격 비교는 더욱 진입장벽이 높다.
- **소비자**는 농산물 소비자가격의 **49.2%가 유통비용**(2023 aT)인 구조에서, 도매 시세가 어떻게 움직이는지 모른 채 장을 본다. KAMIS·도매시장 데이터는 표와 숫자라 일반인이 해석하기 어렵다.

즉 **같은 데이터(도매시장 경락가)가 산지와 소비지 양쪽에서 모두 정보 비대칭의 원인**이다. 오늘장은 이 한 데이터를 양쪽 사용자 언어로 번역한다.

### 정책 정합성
정부는 2023년 11월 **세계 최초 농산물 온라인 도매시장**을 출범시켜 누적 1.2조 원을 거래, **2030년 7조 원·유통비용 10% 절감**을 목표로 유통 디지털 전환을 추진 중이다. 전자송품장·분산출하 정책도 "특정 시장 쏠림 완화 → 가격 안정"을 명시한다. **오늘장의 실수령 라우팅(분산출하 유도)·실시간 시세 투명화는 이 정책 방향과 정확히 일치**한다.

## 2. 솔루션 — 양면(Two-sided) 가격 코파일럿

### 🚜 농가 탭 — 출하 실수령 라우팅 (핵심 기능)
작목·등급·출하지·물량을 입력하면:
1. **전국 12개(확장 32개) 도매시장의 오늘 경락가**를 한 줄에 비교
2. 각 시장까지 **거리×운송단가 + 도매수수료(5~7%)를 차감한 ‘실수령’**으로 줄 세움
3. **"오늘은 ○○시장 출하 시 실수령 1위"** 행동 처방 + 기상 기반 출하 타이밍 배지(강수·고온 시 조기출하 권장)

> KAMIS·농넷이 '가격을 보여주는' 데서 멈출 때, 오늘장은 **'어디로 보내 얼마 더 받아라'는 행동**까지 처방한다. 운송비 차감 실수령 비교는 기존 서비스에 없는 공백이다.

### 🛒 소비자 탭 — 장보기 신호등
- 농축산물 18종을 **평년(직전 5년 동기간) 대비 빨강(비쌈▲)/초록(쌈▼)/회색(보통＝)** 신호등으로 번역
- **오늘의 알뜰 장바구니**(평년比 가장 저렴한 5품목) + 비쌈 품목엔 **같은 부류 대체재 추천**
- 색만으로 판단하지 않도록 **화살표·% 라벨 병기**(색각 접근성)

### 🤖 공통 AI 레이어 (정직한 AI)
- **단기(3~7일) 가격 방향 전망**: 계절성 추세 + 기상 보정 경량 회귀
- **가격 이상탐지**: 30일 분포 대비 ±2.2σ 이탈 품목 자동 감지(급등/급락)
- **⭐ 과장 방지 원칙**: 모든 예측에 **과거 57개 시점 백테스트 정확도(방향 적중률·MAPE)를 함께 공개**한다. (예: 배추 방향 적중률 70.2% vs naive 50%, 평균오차 6.7%) — "AI를 쓴다"가 아니라 "검증된 AI를 정직하게 쓴다".

## 3. 활용 공공데이터 (전부 무료·자동승인/키불필요 — 즉시 재현 가능)

| 데이터셋 | 제공 | 활용 |
|---|---|---|
| **전국 공영도매시장 실시간 경매정보(aT)** `data.go.kr/15141808` | REST, 자동승인 | 시장별 경락가·거래량·등급 (라우팅·신호등 코어) |
| **도매시장 경락가격 표준데이터** `data.go.kr/15029181` | CSV, 키불필요 | 시계열 백필(평년 기준선·백테스트) |
| **기상청 ASOS 일자료** `data.go.kr/15059093` | REST, 자동승인 | 기온·강수 외생변수(출하 타이밍·전망 보정) |
| **주요 채소류 일일가격** `data.go.kr/15087352` | CSV, 키불필요 | 소매가 보조 |

> 본 프로토타입의 경락가는 **헤드리스·무키 환경에서 100% 재현 가능하도록 실제 API 스키마와 호환되는 결정론적 합성 시계열**로 구동된다. 운영 전환 시 `backend/src/liveAdapter` 자리에 발급키(ServiceKey)만 넣으면 **코드 변경 없이 라이브 API로 대체**된다. (정직성 명시 — 데이터를 지어내지 않고, 시연 안정성과 운영 전환성을 동시에 확보)

## 4. 차별성 (베끼지 않고 빈틈을 친다)

| 기존 | 한계 | 오늘장 |
|---|---|---|
| KAMIS / 농넷 | 가격 **조회**에서 멈춤 | 운송비 차감 **실수령 라우팅(행동 처방)** |
| 팜모닝·그린랩스 | 영농일지·자재 커머스 중심 | 출하 의사결정·시세 특화 |
| 물가 알림앱 | 소비자 단면만 | **같은 데이터의 산지↔소비지 양면** |

심사 핵심 반박("KAMIS와 뭐가 다른가?")에 대한 한 줄 답: **"KAMIS는 가격을 보여주고 끝납니다. 오늘장은 같은 경매 데이터로 농가엔 실수령 라우팅, 소비자엔 평년대비 신호를 줘서 한 데이터의 양쪽 끝을 잇습니다."**

## 5. 사업화 (BM)

- **B2C 프리미엄**: 농가 다작목 알림·실수령 시뮬(월 3,900원), 소비자 맞춤 알림
- **B2B (핵심)**: 산지유통센터(APC)·작목반·로컬푸드 직매장 — 회원 농가 출하 최적화 대시보드 연 구독. 운송·물류 플랫폼 공동배차 송출 수수료
- **정책 연계**: 농식품부 산지유통 활성화·온라인도매시장 사업, 지자체 로컬푸드 푸드플랜과 매칭 (데이터가 공공이라 정책 정당성 확보)

전국 32개 공영도매시장 청과 거래 **약 15조 원** 규모가 대상 시장이다.

## 7. 인용 근거 (팩트체크 완료, 출처 확인)

1. 농산물 소비자가격의 **49.2%가 유통비용**(배추·무 64.3%, 월동무 78.1%) — *aT 2023 유통비용조사*
2. 농가 **65세 이상 55.8%**(2024, 사상 첫 55%↑) — *통계청 2024 농림어업조사*
3. 전국 **공영도매시장 32개**, 청과 거래 약 15조 원 — *aT*
4. 시장별 경락가 별도 공시 → 시장 간 가격차 발생 — *KAMIS(aT)*
5. 정부 분산출하 정책으로 "쏠림 완화 → 가격 안정" — *농식품부 전자송품장*
6. 채소·과일은 기상에 따라 급등락(물가 변동성 핵심) — *농업전망 2025·농관원*
7. 온라인 도매시장 2023.11 출범, 누적 1.2조, 정부 2030년 7조·유통비용 10%↓ 목표 — *정책브리핑*

## 8. 기술 구조

```
프론트엔드(Vite + React + Recharts)  ──REST──▶  백엔드(Node Express, 비민감 로직 전용)
 · 농가/소비자 듀얼 탭                            · /api/routing  실수령 라우팅
 · 디자인토큰(농업 그린·tabular·접근성)            · /api/signal   평년대비 신호등
 · 백엔드 미연결 시 번들 스냅샷 폴백               · /api/forecast AI 전망+백테스트
                                                  · /api/anomaly  이상탐지
                                                  · data.go.kr 라이브 어댑터 자리 준비
```
- 개인·민감정보 미수집(시장 가격은 공공데이터). 백엔드는 비민감 집계/계산만 담당.
- `npm run build` EXIT 0, `npm run selftest` ALL PASS 검증.
