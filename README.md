# 오늘장 (ONULJANG)

> 도매시장 실시간 경매 데이터로 **산지(농가)와 소비지(국민)를 잇는 양면 가격 코파일럿**
> 제11회 「농업·농촌 공공데이터+AI」 활용 창업경진대회 · 제품·서비스 개발 트랙 출품작

농가에는 **운송비 차감 실수령 기준 최적 출하 시장**을, 소비자에는 **평년 대비 장보기 신호등**을 제공한다.
하나의 공공데이터(전국 공영도매시장 실시간 경매)를 두 사용자 언어로 번역하는 것이 핵심.

## 구조
```
onuljang/
├── frontend/   Vite + React + Recharts (농가/소비자 듀얼 탭, 디자인토큰, 오프라인 스냅샷 폴백)
├── backend/    Node + Express (비민감 분석 로직: 라우팅·신호·AI전망·이상탐지)
├── shots/      스크린샷(시각 검증)
├── 기획서.md · 발표자료_개요.md · 시연_시나리오.md · 제출_체크리스트.md
```

## 실행
### 1) 백엔드 (포트 8091 — CC_PORT로 분리, agent-core 8848 등과 충돌 방지)
```bash
cd backend
npm install
CC_PORT=8091 npm start        # http://127.0.0.1:8091/api/health
npm run selftest              # 엔진 무결성 자가검증 (ALL PASS)
```
### 2) 프론트엔드 (포트 4173)
```bash
cd frontend
npm install
npm run build                 # EXIT 0
npm run preview               # http://127.0.0.1:4173
# 개발: npm run dev (5173)
```
> 백엔드가 떠 있으면 헤더에 "실시간 경매 연결", 아니면 번들 스냅샷("스냅샷")으로 동작 — **백엔드 없이도 화면이 뜬다**(시연 안정성).

## API
| 엔드포인트 | 설명 |
|---|---|
| `GET /api/health` | 상태 |
| `GET /api/meta` | 시장·작목·등급·출하지·데이터출처 |
| `GET /api/routing?item=&grade=&origin=&qty=` | 시장별 실수령 랭킹 + 처방 |
| `GET /api/signal` | 소비자 평년대비 신호등 |
| `GET /api/forecast?item=` | AI 단기전망 + 백테스트 정확도 |
| `GET /api/anomaly` | 가격 이상탐지(±2.2σ) |
| `GET /api/board` | 시장별 시세 보드 |

- 기상청 ASOS 일자료 `data.go.kr/15059093`
- 주요 채소류 일일가격 `data.go.kr/15087352`

### 데이터 정직성
프로토타입의 경락가는 **실제 API 스키마와 호환되는 결정론적 합성 시계열**이다(헤드리스·무키 환경 재현성 + 시연 안정성). `backend/src/data.js` 상단 주석 참조. 운영 시 `liveAdapter` 자리에 발급키만 넣으면 **코드 변경 없이 라이브 API로 대체**된다. 가격을 임의로 지어내지 않으며, 시장코드/품목분류/기상곡선은 실제 체계를 따른다.

## 개인정보·보안
시장 가격은 공공데이터로 민감정보가 아니다. 개인정보를 수집·저장하지 않으며, 백엔드는 비민감 집계/계산만 수행한다.
