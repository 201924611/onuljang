# 오늘장 — 2차 발표 시연 시나리오 (대본)

> 목표: 5분 내에 "같은 데이터를 양쪽으로 번역한다"는 핵심을 각인. 라이브 데이터로 화면이 살아 움직이는 인상을 준다.
> 준비: 백엔드(8091)·프론트(4190) 기동 후 브라우저 전체화면. 인터넷 없이도 번들 스냅샷으로 동작(시연 안정성).

## 0. 오프닝 (20초)
"농산물은 같은 배추라도 전국 32개 도매시장마다 가격이 다릅니다. 그런데 농가는 늘 가던 시장에만 보내고, 소비자는 그 시세를 모른 채 장을 봅니다. **하나의 도매시장 데이터가 양쪽 모두에게 정보 격차의 원인**입니다. 오늘장은 이 데이터를 양쪽 언어로 번역합니다."

## 1. 농가 탭 — 실수령 라우팅 (90초) ★핵심
1. 작목 **배추 / 등급 상 / 출하지 강원 춘천 / 1,000kg** 선택.
2. 상단 카드 강조:
   - **"오늘의 추천 시장: 안양"** — 실수령 1위
   - **"최적-최악 시장 실수령 격차 39.6%"** → "같은 배추인데 시장 선택만으로 실수령이 40% 차이납니다."
   - **"관성 출하(가락) 대비 +12,200원"**
3. 표를 가리키며: "경락가가 아니라 **운송비·수수료를 뺀 실수령으로 줄을 세웁니다.** 가락이 경락가는 높아도, 거리·수수료를 빼면 순위가 바뀝니다."
4. **출하 처방** 카드: "강수 예보 시 '조기 출하 권장', AI가 3일 방향까지 — 단, **백테스트 적중률을 함께** 보여드립니다."
5. 작목을 **사과**로 바꿔 화면이 즉시 재계산되는 것 시연.

## 2. 소비자 탭 — 장보기 신호등 (70초)
1. 상단 탭 클릭 → 헤드라인이 "오늘, 무엇이 싸고 비쌀까?"로 전환.
2. **신호등 그리드**: "초록은 평년보다 쌈, 빨강은 비쌈. **색맹을 위해 ▲▼ 화살표와 %를 병기**합니다."
3. **알뜰 장바구니**: "오늘 가장 저렴한 5품목을 자동 추천. 비싼 품목엔 같은 부류 대체재를 제안합니다."
4. 비쌈 품목 클릭 → 우측 **AI 단기 전망 차트** 갱신.

## 3. AI의 정직함 (40초) ★차별점
1. 전망 차트 하단 지표를 가리키며: "**방향 적중률 70.2%, naive 50% 대비. 평균오차 6.7%. 검증표본 57개.** 저희는 예측을 띄우기만 하지 않고 **얼마나 맞는지를 같이 공개**합니다. 변동성 큰 품목(딸기 77%)은 잘 맞고, 안 맞는 품목(감귤 42%)도 숨기지 않습니다."
2. 상단 **AI 이상감지** 칩: "평소 분포를 벗어난 급등·급락을 자동 포착합니다."

## 4. 클로징 (20초)
"오늘장은 **하나의 공공데이터(도매시장 실시간 경매)를 산지와 소비지 양쪽 끝까지** 연결합니다. 전부 무료·자동승인 데이터라 내일 바로 운영 가능하고, 정부의 유통 디지털화·분산출하 정책과도 정확히 맞물립니다. 감사합니다."

## 예상 Q&A
- **Q. KAMIS와 차이?** → "KAMIS는 가격 조회까지. 저흰 운송비 차감 실수령으로 '어디로 보내라'를 처방하고, 같은 데이터를 소비자 신호등으로도 번역합니다."
- **Q. 데이터가 합성 아닌가?** → "시연 안정성을 위한 결정론적 합성이지만 **실제 API 스키마와 100% 호환**, 발급키만 넣으면 코드 변경 없이 라이브 전환됩니다. 가격을 지어낸 게 아니라 재현 가능한 데모 레이어입니다."
- **Q. 예측 신뢰도?** → "그래서 모든 예측에 백테스트 정확도를 강제로 함께 표시합니다. 못 맞히는 구간도 공개합니다."
- **Q. 운송비 산정 근거?** → "거리(haversine)×화물 일반운임 근사 + 취급비. 실서비스는 실제 운임 테이블·공동물류 단가로 정교화합니다."
